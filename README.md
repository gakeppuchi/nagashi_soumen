流しそうめんのゲームです。<br>
制作著作　がけっぷち<br>
Copyright (c) 2021 gakeppuchi<br>

## How to play<br>
ハードルを避けながらポイントを獲得してハイスコアを目指そう。<br>
開始時にライフは３つありますが、ミスをするとライフが１つ減ります。ライフが０になるとゲームオーバーです。<br>

#操作<br>
キーボード１：ジャンプ<br>
キーボード２：右に移動<br>
キーボード３：左に移動<br>
キーボード４：スタート<br>
