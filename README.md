## How to play<br>
ハードルを避けながらポイントを獲得してハイスコアを目指そう。<br>
開始時のライフは３つ。ミスをするとライフが１つ減り、ライフが０になるとゲームオーバーです。<br>

### 操作<br>
キーボード入力でキャラクターを操作します。<br>
キーボード１：ジャンプ<br>
キーボード２：右に移動<br>
キーボード３：左に移動<br>
キーボード４：スタート<br>
